# Phase 999.1 Plan 01: Resumo Executivo (Qulture Rocks) — Summary

**One-liner:** Prompt refatorado com estrutura D-03 por responsável + bloco QR destacado no ArtifactCard com botão de cópia.

## What Was Built

### Task 1: Refinar prompt resumo_executivo_rh no Pass 2

O campo `resumo_executivo_rh` no prompt `1on1-deep.prompt.ts` foi reescrito para seguir a estrutura D-03/D-04/D-05 definida no CONTEXT.md. A nova instrução:

- Separa ações por responsável em seções distintas (Ações do liderado / Ações do gestor / Próximos passos)
- Define formato de bullet explícito: `• [Nome]: [ação concreta]`
- Instrui Claude a omitir seções vazias (não gerar bullets sem conteúdo)
- Inclui cláusula `OMITIR obrigatoriamente` listando campos inadequados para RH

**Commit:** `c327717`

### Task 2: Bloco QR no ArtifactCard + botão Copiar

O `ArtifactCard` em `PersonView.tsx` recebeu:

- **`extractQRSummary(content)`**: função helper que usa regex `/##\s*Resumo Executivo \(Qulture Rocks\)([\s\S]*?)(?=\n##\s|\s*$)/` para extrair o texto da seção
- **`copyQR(text, e)`**: função assíncrona que usa `navigator.clipboard.writeText()` e controla estado `copied` para feedback visual
- **Bloco QR condicional**: aparece apenas quando `a.tipo === '1on1'` E `extractQRSummary` retorna conteúdo — posicionado acima do `<MarkdownPreview>`
- **Botão "Copiar para QR"**: mostra "Copiado!" por 2 segundos após cópia, usando tokens de cor do design system

**Commit:** `8f102c9`
